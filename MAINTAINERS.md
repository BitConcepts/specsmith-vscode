## Scope

This repository contains the **VS Code extension** for the specsmith AEE Workbench.
It is a pure TypeScript UI layer — all AI reasoning, tool execution, and governance logic
lives in the [specsmith Python CLI](https://github.com/BitConcepts/specsmith).

Maintainers are responsible for:
- Reviewing and merging pull requests
- Triaging issues and security reports
- Releasing new extension versions (`.vsix`)
- Keeping the extension compatible with new specsmith CLI releases
- Ensuring dependency health (Dependabot, npm audit)

## Becoming a Maintainer

Contributors who demonstrate sustained, high-quality contributions may be invited to
become maintainers. Reach out via [GitHub Issues](https://github.com/BitConcepts/specsmith-vscode/issues)
or email.
